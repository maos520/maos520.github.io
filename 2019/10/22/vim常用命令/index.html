vim常用命令

Posted on 2019-10-22 Edited on 2020-01-05 In Linux , vim Views:

一、前言

​ 本篇博客主要记录一些我在使用vim中常用的快捷键

二、基本操作

​ vim的快捷很多，但是常用的并不是特别多

  1. 移动

    • h,j,k,l基本的上下左右移动
    • gg 文章开头
    • GG 文章结尾
    • ^ 行首
    • $ 行尾

  2. 查找和替换

    • gd 查找光标所在变量定义处
    • * 向后查找单词
    • # 向前查找单词
    • f / F / t / T 行内差字母
    • ； 重复 f 命令
    • / + 查找内容
    • ：%s/old/new/gc 替换 c 表示要确认
    • n下一个匹配

  3. 删除 、复制和粘贴

    • dd

    • dw

    • x

    • y

    • p

    • "{寄存器}y 将选中内容复制到寄存器里

    • “{寄存器}p 将寄存器的内容粘贴

  4. 标记 mark

    • 在normal模式下 m+{a-z}用字母a-z标记现在光标的位置

    • `+{mark}跳转到标记处；'+{mark}跳转到标记处行首

    • :marks显示标记列表

    • 常用特殊标记

      1. '跳转前的位置
      2. "最后修改的位置

三、多文件操作

  1. buffer相关快捷键

    • :edit 编辑
    • :new创建
    • :bn 下一个
    • :bp上一个
    • :bd关闭当前
    • :ls b显示
    • :b 数字跳到相应buffer

  2. 窗口相关快捷键

    • :sp + 文件 水平分割窗口
    • :vsp +文件 垂直分割窗口
    • ctrl-w h 跳转到左边的窗口
    • ctrl-w k 跳转到上面的窗口
    • ctrl-w l 跳转到右边的窗口
    • ctrl-w j 跳转到下面的窗口
    • ctrl-w c 关闭当前窗口

四、 批量操作

  1. 命令宏

    qx+命令串+q 将会录制命令串，然后同数字+@+x将会重读执行录制的命令串，结合移动(h,j,k,l)，查找（f t）,修改命令，会变得很强大。

  1. ctrl + v 进行块选中模式 ，选中相应的块 通过大写的 I 进入插入模式，这时对首行修改完之后按ESC，修改的操作会自动加在之前选中的其他行。

五、括号操作

  1. v/y/d/c+i/a+{/[/(，其中v,y,d,c即我们常用的选择，复制，删除和修改，i即inside的意思，对括号里面的

    内容进行操作，不包括括号；a即at的意思，对括号里的内容包括括号进行操作。

  2. %在表示范围时，表示整个文档，除此之外，还有匹配括号的作用。

其他

  1. ctrl + s ：vim会停止向终端输出，像死掉了一般！！！

  2. ctrl + q ：vim会退出停止状态！！！！

  3. vmap,imap,imap前首字母分别代表不同的模式，分别代表虚拟模式，普通模式，插入模式。nore代表非递归。

  4. gu， gU， guw ，~大小写转换相关命令

  5. :sh ， ctrl+d 进入命令行以及返回vim

  6. :!command调用shell命令, : r !command将结果插入到下一行 ，e.g. :r !data将当前日期插入到下一行
